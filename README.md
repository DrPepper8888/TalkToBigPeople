# 与智者对话 - 微信小程序

> 和你喜欢的伟人/智者对话，感受他们的智慧，AI严格按照你设定的规则模仿他们的说话风格和思维方式。

## 功能

- ✅ 预置多位智者：冯·诺依曼、爱因斯坦、乔布斯、鲁迅、苏轼
- ✅ **支持用户创建自定义名人**——按照标准规则填入名人名言/作品/传记，AI就能完美角色扮演
- ✅ 作品自动带注释翻译：诗人的诗词可以附上原文注释和白话翻译，聊天时自动说明
- ✅ 支持流式输出，聊天丝滑流畅
- ✅ 对接 one-api 统一接口，支持各种开源大模型，个人可免费部署
- ✅ 微信小程序端 + Node.js 后端，完全符合质量门禁标准

## 人物数据标准规则

每个智者/名人包含四类标准化数据：

| 数据项 | 说明 |
|--------|------|
| **名人名言** | 这个人的口头禅、经典语录，聊天时AI会自然穿插 |
| **核心作品** | 著作/诗词/文章，可以保留原文，并加上注释和现代文翻译 |
| **人物传记** | 生平、核心思想、性格特点，帮助AI更贴近角色 |
| **系统提示词** | 自动生成，不需要手动写 |

当聊到作品时，AI会自动按照你的要求：引用原文 → 小括号放注释 + 翻译，非常清晰。

## 部署

### 后端
```bash
cd server
npm install
# 配置环境变量
export OPENAI_BASE_URL=your-one-api-url
export OPENAI_API_KEY=your-api-key
export PORT=3000
npm start
```

### 小程序
1. 打开微信开发者工具，导入 `miniprogram` 目录
2. 修改 `app.js` 里的 `apiBaseUrl` 改成你的后端地址
3. 上传发布即可

## 环境变量

后端：
- `OPENAI_BASE_URL`：你的 one-api 地址，默认 `http://localhost:3000/v1`
- `OPENAI_API_KEY`：你的 API key
- `PORT`：监听端口，默认 3000
- `MODEL`：模型名称，默认 `gpt-3.5-turbo`

## 使用

### 内置名人直接用
打开小程序，点击想要对话的名人，直接开始聊天。

### 创建自定义名人
1. 点击首页"创建自定义人物"按钮
2. 按照表单填写：
   - 人物名称（必填）
   - 简短描述（必填）
   - 人物传记/核心思想
   - 名人名言（每行一句）
   - 核心作品（可以添加多篇，填写标题/原文/翻译/注释）
3. 点击创建，创建完成就能直接对话了
4. 不需要的自定义名人可以左滑删除
